📍 CSS Position Etkinliği

Position: relative, absolute ve fixed kullanarak öğren

Teknoloji Toplantığı 2026

Hoş geldiniz! Bu sayfada CSS position özelliklerini pratik ederek öğreneceksiniz. Sayfada göreceğiniz her element farklı position değerleri kullanılarak konumlandırılmıştır.

Position properties:

  • position: relative - Ana kapsayıcı için kullanılır
  • position: absolute - Relative parent içinde konumlandırılan öğeler
  • position: fixed - Scroll sırasında sabit duran navbar

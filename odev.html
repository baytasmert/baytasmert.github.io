Verilen benzer bir duzenle basit bir web sayfasi olusturun. CSS'in temel ozelliklerini (display, width, height, margin, padding, border, selector) uygulayin. Sadece HTML5 + CSS3 kullanin, hazir framework kullanmak yasak (Bootstrap, Tailwind vb.).

Icerik

Bu alanda ornek icerik metinleri bulunmaktadir. CSS ozellikleriyle bu metinler arasinda bosluklar (line-height) ve hizalamalar (text-align) duzenleriz.

Once CSS ile kutulari / alanlari yan yana getirmeyi, ardindan alt alta yerlestirmeyi ogrendik. Simdi ise CSS secicilerini kullanarak grup ve hiyerarsik secimler yapmayi gorecegiz.

Yan Menu / Bilgi

Bu alan yan menu veya bilgi kismidir. CSS ile genislik (width), yukseklik (height), kenar bosluklari (margin/padding) ve kenarlik (border) ozelliklerini kullanarak duzenleyebiliriz.

Ogrendiklerimizi hiyerarsik ve grup secicilerle uygulayip farkli bolumlere stil verebiliriz.

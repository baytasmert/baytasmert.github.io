📋 Ödev Açıklaması

Bu ödevde Microsoft Azure platformu üzerinde öğrenci hesabı oluşturarak, örnek bir web sayfasını Azure ortamında yayınlamayı öğreneceksiniz. Bulut bilişim platformlarını tanıyacak ve web sitesi yayınlama mantığını uygulamalı olarak göreceksiniz.

🎯 Öğrenme Hedefleri

Bulut Bilişim

Azure gibi bulut platformlarının temel özelliklerini ve faydalarını anlamak

Azure Platformu

Microsoft Azure portalını kullanmayı ve kaynak oluşturmayı öğrenmek

Web Yayınlama

Web sitesi yayınlama mantığını uygulamalı olarak görüp uygulamak

📝 Yapılacak İşlemler

1️⃣ Azure for Students Hesabı Oluşturma

Azure for Students linkine giderek öğrenci e-posta adresinizle kayıt yapın.

2️⃣ Azure Portal'a Giriş

Azure Portal'a giriş yaparak ana sayfaya erişin.

3️⃣ Örnek Web Sayfası Oluşturma

Bilgisayarınızda AzureOdev klasörü açın ve içinde index.html dosyası oluşturun. Dosyanızda şunlar olmalı:

  • Ad Soyad
  • Öğrenci Numarası
  • Bölüm Bilgisi
  • Kısa Tanıtım Yazısı

4️⃣ Web Sitesini Azure'da Yayınlama

Azure Portal'dan Storage Account oluşturup, Static Website özelliğini açarak web sitesini yayınlayın.

📦 Adım Adım Azure Yayınlama

Adım 1: Storage Account Oluşturma

  1. Azure Portal'a girin
  2. Arama kutusuna "Storage accounts" yazın
  3. Yeni Resource Group oluşturun (örn: azure-odev-rg)
  4. Küçük harflerle benzersiz bir Storage account adı yazın
  5. Performance: Standard, Redundancy: LRS seçin
  6. Create butonuna basın

Adım 2: Static Website Özelliğini Açma

  1. Storage Account içerisine girin
  2. Sol menüden: Data management → Static website
  3. Static website: Enabled seçin
  4. Index document name: index.html yazın
  5. Save butonuna basın

Adım 3: HTML Dosyasını Yükleme

  1. Sol menüden: Containers → $web container
  2. Upload butonuna basın
  3. index.html dosyasını seçin
  4. Upload butonuna basın

Adım 4: Web Sitesini Görüntüleme

  1. Static website sayfasına geri dönün
  2. Primary endpoint linkini kopyalayın
  3. Tarayıcıda açarak çalışmasını kontrol edin

📤 Teslim Edilecekler

Aşağıdaki bilgileri rapor halinde sunun:

  • Ad Soyad
  • Öğrenci Numarası
  • Kullanılan Azure servisi
  • Yayınlanan web sitesi bağlantısı (Primary endpoint URL)
  • Azure portal ekran görüntüsü
  • Web sitesinin çalışır hâlinin ekran görüntüsü
